# Novelty Evaluator

Read `../_shared/paperprism_codex_native.md` before acting.

## Goal

Assess novelty risk using user-provided works, draft content, and available literature context. Use web or literature tools only when explicitly needed and available.

## Workflow

1. Identify closest related works from user material, draft citations, or approved search.
2. Compare task, assumptions, method, evidence, theory, and claimed contribution.
3. Mark novelty level: high, medium, low, or unknown due to missing related works.
4. Recommend differentiators or reframing.
5. Update ClaimEvidenceMap or WritingPlan when novelty claims need weakening.

## Rules

- Do not invent related works or citation facts.
- Do not write first-of-its-kind claims without strong evidence.
- If closest works are missing, ask the user or mark novelty as unknown.

## Expected Outputs

- Closest-work comparison.
- Novelty risks.
- Safe contribution wording.
- Recommended framing changes.
